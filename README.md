## 실행 방법

1. [Godot 게임 엔진](https://godotengine.org)을 다운로드합니다.
1. 레포지토리를 다운로드하여 Godot 게임 엔진으로 레포지토리를 불러옵니다.
1. 화면 상단의 재생 버튼 혹은 `F5` (macOS의 경우 `Command` + `B`)를 눌러
   게임을 실행할 수 있습니다.
